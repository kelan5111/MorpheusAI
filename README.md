<h1>MorpheusAI</h1>

<h2>AI Storyteller</h2>
<p1></p1>LLM which pulls the user into a deep and descriptive story, playing both characters and the narrator.
An immersive interactive storytelling experience where you play as the main character. The AI acts as the narrator and controls all other characters with distinct voices and personalities.</p1>
